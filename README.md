# ☕ Pa' Picar Coffee – Website

🌐 Live: https://papicarcoffe.xo.je/

I created this modern, responsive website for a local coffee shop in Almería, Spain after noticing they didn’t have a website to represent them online.

I liked the place and decided to build a clean, simple website that showcases their café, menu, and overall vibe.

## ✨ Features
- Responsive design (mobile, tablet, desktop)
- Smooth animations and clean UI
- Menu and gallery sections
- Contact info with map and click-to-call

💡 I build modern websites for local businesses to help them stand out online.
